MEGA

Apa yang disukai

  • Banyak ruang penyimpanan.

  • Share folder publik.

  • Mobile dan desktop aplikasi.

  • Bersih dan modern tampilan.

Apa yang tidak disukai

  • Lambat untuk mempersiapkan file untuk di-download.

  • keterbatasan bandwidth.

  • Tidak ada canggih opsi berbagi link.

Dapatkan 50 GB penyimpanan awan gratis dengan MEGA. MEGA menawarkan enkripsi aman end-to-end untuk membantu memerangi pelanggaran privasi.

Anda dapat melihat dan memperbarui file MEGA Anda melalui browser, klien sinkronisasi desktop, atau aplikasi mobile.

Dapatkan MEGA untuk  iOS ,  BlackBerry ,  Android , Windows Phone , atau Windows / Mac / Linux .

pCloud

Apa disukai

  • Lebih penyimpanan gratis daripada kebanyakan layanan.

  • Dapatkan ruang tambahan.

  • Streaming multimedia.

  • Tidak ada speed atau batas ukuran file.

Apa yang tidak disukai

  • Lacks keamanan file sharing.

10 GB penyimpanan online gratis yang ditawarkan oleh pCloud, dengan teman arahan dan langkah-langkah bonus lainnya melompat ke sebuah kemungkinan 20 GB.

Seorang klien desktop dapat diinstal untuk sistem operasi Windows, Mac, dan Linux. Ada juga aplikasi mobile yang sangat intuitif dan mudah digunakan.

pCloud mendukung penambahan file dari URL terpencil di samping file tunggal dan upload folder keseluruhan melalui browser. Aplikasi mobile dapat meng-upload foto dan video langsung ke rekening pCloud Anda.

Kedua file dan folder dapat berdua akan berbagi dengan non-pengguna. folder bersama bahkan dapat didownload sebagai arsip ZIP.

Juga, pCloud memungkinkan meng-upload link yang akan dihasilkan untuk folder sehingga siapapun dapat mengirim file ke akun Anda. Sebuah alamat email yang unik juga dihasilkan yang memungkinkan Anda (atau siapa pun) file email langsung ke folder di akun pCloud Anda.

Selain di atas, Anda dapat backup foto dari akun media sosial Anda seperti Facebook, Instagram, dan Picasa.

Dapatkan pCloud untuk iOS , Android , Windows Phone , dan lebih .

FlipDrive

Apa disukai

  • Jumlah berguna penyimpanan.

  • Cross-platform aplikasi mobile.

  • Unik izin berbagi file.

Apa yang tidak disukai

  • Tidak ada akses publik ke folder bersama.

  • Kecil file upload batas ukuran.

  • Tidak ramah antarmuka situs.

Pengguna baru FlipDrive mendapatkan 10 GB penyimpanan awal dengan kesempatan lebih banyak untuk setiap teman mereka dapat merujuk untuk mulai menggunakan FlipDrive.

Ada aplikasi mobile - untuk iOS , Android , Windows & Windows Phone  - yang dapat digunakan untuk meng-upload file ke FlipDrive, tapi website bekerja, juga. Jika Anda menggunakan versi web, Anda dapat meng-upload seluruh folder menggunakan website, yang bagus.

Anda dapat berbagi file dengan siapa pun, bahkan jika mereka tidak memiliki account FlipDrive. Namun, mengakses bersama  folder memerlukan akun FlipDrive terdaftar. Penerima dapat diberikan menulis, mengedit, menghapus, dan izin share di samping untuk membaca hak untuk folder.

Anda dapat membuat hingga 10 link bersama dan meng-upload file yang besar seperti 25 MB. 

Situs web FlipDrive tidak yang paling mudah untuk digunakan bila dibandingkan dengan sebagian besar layanan penyimpanan awan lainnya, yang menyedihkan, tapi masih berhasil berfungsi jika Anda dapat bekerja melalui itu.

google Drive

Apa disukai

  • situs web super user-friendly.

  • Kolaborasi dan file sharing.

  • Banyak penyimpanan gratis.

  • Desktop dan aplikasi mobile.

Apa yang tidak disukai

  • Penyimpanan bersama dengan layanan Google lainnya.

  • Bare desktop yang sinkron client.

Google Drive adalah layanan penyimpanan cloud yang ditawarkan oleh Google. Setiap pengguna baru mendapatkan 15 GB ruang bebas.

Total penyimpanan benar-benar berbagi dengan layanan Google lainnya, seperti Gmail dan Google+ Foto. Jika Anda tidak menggunakan layanan lainnya, Anda dapat memanfaatkan hampir semua ruang untuk Google Drive sendiri.

Seorang klien sync dapat diinstal untuk pengguna desktop, yang mendukung folder dan upload file. Ada juga aplikasi mobile yang dapat mengirim foto dan video ke Google Drive.

Folder dan file dapat dibagi dengan pengguna Google tertentu melalui alamat email mereka atau siapa pun dengan link publik. Anda juga dapat membuat file pandangan-satunya, yang mencegah seseorang dari mendownloadnya.

Google Drive membuatnya benar-benar sederhana untuk berkolaborasi dengan orang lain dengan memungkinkan komentar dan membiarkan file dibuka dan diakses saya lebih dari satu orang pada satu waktu.

Dapatkan Google Drive untuk iOS , Android , atau Anda komputer .

Apa yang tidak disukai

Degoo

Apa disukai

  • penyimpanan yang lebih daripada kebanyakan layanan.

  • Aplikasi untuk mobile dan desktop.

  • meng-upload file secara otomatis.

  • Beberapa cara untuk mendapatkan lebih banyak ruang penyimpanan gratis.

Apa yang tidak disukai

  • Bekerja lebih seperti layanan backup.

  • Tidak ada dukungan hard drive eksternal.

  • tidak dapat melihat file yang tersimpan di website.

Degoo memberikan sebuah besar 100 GB penyimpanan gratis untuk siapa saja yang membuat akun. Anda dapat menggunakan akun penyimpanan awan pada komputer Anda melalui Program desktop yang , atau mengakses file Anda pada pergi dengan aplikasi mobile untuk iOS dan Android .

Jika Anda memiliki banyak teman-teman yang ingin menggunakan Degoo untuk menyimpan file mereka, Anda bahkan bisa mendapatkan 500 GB  lebih  dengan merujuk mereka untuk membuat akun. Anda mendapatkan 3 GB untuk setiap orang yang bergabung melalui referral Anda. Anda juga dapat menonton iklan untuk mendapatkan lebih banyak ruang online gratis (1,2 GB per hari).

Saya sudah ditempatkan Degoo di bagian paling bawah dari daftar ini meskipun kapasitas 100 GB yang besar karena fungsinya sedikit lebih seperti layanan backup daripada program penyimpanan awan.

Anda pasti dapat memilih file mana yang didukung tetapi backup dilakukan secara otomatis setiap 24 jam untuk menjaga hal-hal diperbarui, yang membuatnya bekerja lebih seperti layanan backup gratis.

Karena Degoo sedikit berbeda dari ini layanan penyimpanan awan lainnya, Anda harus menggunakannya seperti ini: memilih folder di komputer Anda yang akan menampung semua didukung file Anda, atau membuat yang baru. Meletakkan segala sesuatu ke dalam folder yang Anda ingin menyimpan secara online, dan kemudian pastikan Degoo berjalan untuk membuat cadangan file secara online.
